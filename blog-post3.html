Automating Order Tracking & Daily Spreadsheet Backups

1. Introduction: The Problem

Before automation, employees manually tracked new orders and added them to a spreadsheet. This process was inefficient and led to several issues:

  • Employees had to juggle order tracking with other tasks, causing delays.
  • Orders placed outside business hours had to be manually entered the next day, which took hours during busy seasons.
  • Every morning, employees manually copied and renamed the spreadsheet to create a backup, adding another time-consuming step.

2. The Solution

To eliminate these inefficiencies, I built a Google Apps Script Web App that automates order tracking and spreadsheet backups. The system works as follows:

  • Receives webhooks from WooCommerce API when a new order is placed.
  • Automatically updates the Google Spreadsheet, eliminating the need for manual tracking.
  • Runs continuously, even outside business hours, ensuring real-time updates.
  • At midnight, it creates a fresh copy of the spreadsheet, renaming it accordingly.
  • Sends email notifications to employees with the updated table link for seamless workflow continuation.

Tech Stack: Google Apps Script, Google Sheets API, WooCommerce API, JavaScript, Google Mail API (for email notifications), Postman (for API testing).

3. Results & Impact

  • ✅ Eliminated manual tracking, allowing employees to focus on higher-value tasks.
  • ✅ Saved hours of work daily, especially during peak seasons.
  • ✅ Reduced human errors, ensuring accurate order tracking and backup processes.
  • ✅ Provided real-time updates, ensuring orders are logged immediately, even outside working hours.

4. Challenges & Learnings

One of the main challenges was extracting only the necessary order data to fit into the spreadsheet correctly. Through debugging and fine-tuning the WooCommerce API response, I was able to filter and structure the data efficiently for seamless integration.

5. Conclusion

This automation revolutionized the order tracking process, saving valuable work hours while improving efficiency and accuracy. Future improvements could include integrating advanced reporting to provide employees with insights into order trends and performance tracking.
